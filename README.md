## Install

Install dependencies
```
npm i
```

## Run

**Develop**
> Starts node server with nodemon on 8080 and listen for .scss and .js changes, **don't forget to comment out uglify on webpack.config.js**

```
gulp dev
```
**Build**
> Compiles .scss and .js into bundles in *dist/*

```
gulp build
```
